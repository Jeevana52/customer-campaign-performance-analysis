# Customer Campaign Performance Analysis Dashboard

An interactive analytics dashboard built using Python, Streamlit, Plotly, and SQL to analyze customer engagement in banking campaigns.

## Features
- KPI metrics
- profession-based response analysis
- campaign outcome distribution
- trend analysis
- downloadable filtered report

## Tools Used
- Python
- Streamlit
- Plotly
- Pandas
- SQL

## Dataset
Portuguese Bank Marketing Dataset

## Run Locally
streamlit run app.py
